# Question 133A - HQ9+

## Problem

Given a program written in the HQ9+ language, determine whether executing it will produce any output.

The commands that generate output are:

* `H`
* `Q`
* `9`

Print:

* `YES` if the program contains at least one of these commands
* `NO` otherwise

---

## Initial Thought Process

* We do not need to simulate the program.
* The only thing that matters is whether any output-producing command exists.
* So we just need to check if the string contains `H`, `Q`, or `9`.

---

## Insight

* If the program contains at least one of `H`, `Q`, or `9`, it will produce output.
* Otherwise, no output is generated.

---

## Algorithm

```text
READ program

IF program contains "H"
    PRINT "YES"
ELSE IF program contains "Q"
    PRINT "YES"
ELSE IF program contains "9"
    PRINT "YES"
ELSE
    PRINT "NO"
```

---

## Edge Cases

* Program contains only one output command → `YES`
* Program contains multiple output commands → `YES`
* Program contains only `+` characters → `NO`
* Output command appears anywhere in the string → `YES`

---

## Time and Space Complexity

* **Time Complexity:** `O(n)`
* **Space Complexity:** `O(1)`

---

## Java Code

```java
import java.io.*;

public class Main {
    public static void main(String [] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String instruction = br.readLine();

        if (instruction.contains("H") ||
            instruction.contains("Q") ||
            instruction.contains("9")) {
            System.out.println("YES");
        } else {
            System.out.println("NO");
        }
    }
}
```
